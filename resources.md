## General resources

### Textbooks

- Kosuke Imai, *Quantitative Social Science: An Introduction* (Princeton UP, 2017)
- Diez, Cetinkaya-Rundel, and Barr, *OpenIntro Statistics* (OpenIntro, 4th ed, 2019)
- Francisco Urdinez and Andres Cruz, *R for Political Data Science: A Practical Guide* (CRC, 2020)
- Ismay, Kim, and Valdivia, [*Statistical Inference via Data Science: A ModernDive into R and the Tidyverse*](https://moderndive.com/) (Online/CRC, 2025)

### Git and Version Control

- [Happy Git with R](https://happygitwithr.com) - Essential guide for R users
- [Git Exercises](https://gitexercises.fracz.com/) - Interactive exercises to practice Git
- [GitHub's Official Guides](https://docs.github.com/en/get-started)
- [Pro Git Book](https://git-scm.com/book/en/v2) (free online)

### R Resources

- Wickham, Cetinkaya-Rundel, and Grolemund, [*R for Data Science*](https://r4ds.hadley.nz/) (Online/O'Reilly, 2nd ed, 2023)
- Arel-Bundock, Greifer, and Heiss, [*Model to Meaning: How to Interpret Statistical Models Using marginaleffects for R and Python*](https://marginaleffects.com/) (online, 2025)
- Rodrigues, [*Building reproducible analytical pipelines with R*](https://raps-with-r.dev/) (online, 2023)

### Causal Inference

- Nick Huntington-Klein, [*The Effect: An Introduction to Research Design and Causality*](https://theeffectbook.net/) (Chapman and Hall/CRC, 2021)
- Scott Cunningham, [*Causal Inference: The Mixtape*](https://mixtape.scunning.com/) (Yale University Press, 2021)
